Creative CyberSolutions was founded in 2009 by Brian Bird. Brian has over 20 years’ experience as a professional software developer and Computer Science instructor. A team of other software developers work with Brian on software projects as needed.

Our expertise includes:

  • .NET development (C# and Visual Basic)
  • Java Development including ServerPages, Servlets, and Spring MVC
  • SQL programming (T-SQL, PL-SQL and MySQL)
  • Web development with JavaScript, HTML 5, CSS 3 and ASP.NET (Web Forms and MVC)
  • Mobile Development for iOS and Android
  • Device and instrument control with C and C++

The company is based in Springfield, Oregon, but we can serve clients anywhere that the internet reaches - which is just about everywhere!
